# The Objective

This course shows you how to build a quiz using the basic features of Vue 3. We will leave the
visual aspects of the site iteself to your artistic ambitions, but we have some requirements as to the functionality
we want the site to deliver:

1. Use Vue 3 as a library
2. Pre define questions and answers for quiz
3. Get leaderboard from a json file
4. Track points for the person taking the quiz
5. Make use of components, vue-router and vuex

### What will you be learning

* Introduction to Vue
* The Vue CLI
* Directives
* Declarative Rendering
* How to send data to components through props
* Setup watchers that react on state change
* Use computed properties to alter data before it's displayed
* Building logical methods
* How life cycle hooks work
